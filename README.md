# Care Connect

## Project Overview
Care Connect is a web application created to raise awareness for cancer and help users find local support resources during their journey.

This project was built as part of a team during a hackathon.

---

## My Contribution
- Developed and refined frontend components in React and TypeScript
- Styled and implemented UI using Tailwind CSS and shadcn-ui
- Added interactive features like tabs, dynamic content loading, and responsive layouts
- Collaborated with teammates to integrate frontend functionality and deliver a working demo

## Getting Started

### Prerequisites
- Node.js
- npm  
(Recommended: install via `nvm`)

### Installation
Run the following commands in your terminal:

    npm install
    npm run dev

### Usage
- Open the local server address shown after running `npm run dev` 
- Explore available features like "help center", "awareness", and "emergency support"
